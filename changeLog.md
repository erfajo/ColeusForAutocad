### History of ColeusForAutocad
This software has had many names during its lifetime before the current name, Coleus (ColeusforAutocad). Previously, it was known as DanEDU Layers, CadManager, and Dicapro.  
  
  
2026.0415.0  
- Added installation for Autocad 2027.  
  
2025.0901.0  
- The tool is renamed to "ColeusForAutocad".  
- Support for STB layers has been reestablished and still includes CTB layers.  
  
2025.0401.0  
- Added installation for Autocad 2026.  
  
2025.0301.0  
- Added installation for Autocad 2025.  
  
2023.0401.0  
- Added installation for Autocad 2024.  
  
2022.0406.0  
- Added installation for Autocad 2023.  
  
2021.0415.0  
- Added installation for Autocad 2022.  
  
2020.0415.0  
- Added installation for Autocad 2021.  
  
2019.0420.0  
- Added installation for Autocad 2020, installation for Autocad 2016 removed.  
  
2018.0420.0  
- Added installation for Autocad 2019, installation for Autocad 2015 removed.  
  
2017.0420.0  
- Added installation for Autocad 2018, installation for Autocad 2014 removed.  
  
2016.0801.0  
- Added installation for Autocad 2017, installation for Autocad 2013 removed.  
  
4.0.2     2015-07-01  
- Added installation for Autocad 2016.  
  
4.0.1     2014-12-07  
- Only support for CTB layers.  
- Everything is renamed to DanEDU Layer and Object Structure.  
  
4.0.0     2014-11-25
- All previous versions assembled in one installation and renamed to "DanEDU".  
  
3.42.0     2014-08-25  
- 1st version for AutoCAD 2013, 2014, 2015, primarily upgraded from version 3.32.0.  
  
3.32.0     2009-04-01  
- 1st version for AutoCAD 2008, 2009, 2010, primarily upgraded from version 2.01.0.  
- Support for Bips Layer and Object Structure (Bips2005 and Ibb2000).  
  
2.02.0     2005-06-01  
- 1st version for AutoCAD (all versions), primarily upgraded from version 1.01.0.  
- Only support for Bips Layer 2005.  
  
1.02.0     2004-08-16  
- Version for AutoCAD 2000-2002, 2004, ADT2004, ADT2005.  
- Only support for Ibb Layer 2000.  
  
  
This solution is the continued work from the Dicapro solution.
